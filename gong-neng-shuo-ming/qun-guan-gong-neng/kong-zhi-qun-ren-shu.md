## ①控制群人数

 这里可以将群人数控制在您想要控制的范围

{% hint style="danger" %}
检查间隔如果设置的过小会导致多次踢出以至于有冻结风险  
请注意规范操作
{% endhint %}

{% hint style="warning" %}
踢前私聊有一定几率会导致陌生人情况下发送出去对话解除
{% endhint %}

{% hint style="success" %}
如果您使用此功能误踢成员，可以在[QQ恢复](http://huifu.qq.com)中登录群主账号进行恢复操作
{% endhint %}
